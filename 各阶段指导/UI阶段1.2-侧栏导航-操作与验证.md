# UI 阶段 1.2 侧栏导航 — 操作与验证指南

本文档面向执行与验收，按「阶段要达成什么 → 需要准备什么 → 如何验证」说明 **UI 设计计划** 阶段 1.2（侧栏导航）的完整流程。阶段 1.2 在 1.1 侧栏 + 主内容区基础上，落实**当前页高亮**与设计系统一致、**品牌/标题**与 design token 对齐；侧栏导航项（首页、设置、对比）可见且点击跳转正确。「创建任务」入口本阶段不要求增加，保留首页内按钮即可。

---

## 一、阶段 1.2 要达成什么

- **导航项在侧栏**：首页（任务列表）、设置、对比；点击可正确跳转；可根据需要增加「创建任务」入口，本阶段**不要求**增加，保留首页内「创建任务」按钮即可。
- **当前页高亮**：根据当前路由在侧栏内高亮对应导航项；高亮样式与设计系统一致（主色或 token 控制的选中样式）。访问 `/` 时「首页」高亮，`/settings` 时「设置」高亮，`/compare` 时「对比」高亮，`/tasks/:id` 时「首页」高亮（任务详情无独立侧栏项）。
- **品牌/标题**：在侧栏顶部保留「BIM 标书生成」或产品名，样式使用 design token（如 `fontSizeHeading5`、`fontWeightStrong`、`colorText`）；见 [frontend/src/theme/README.md](../frontend/src/theme/README.md)「侧栏与导航（阶段 1.2）」。
- **交付物**：侧栏导航实现完成；切换路由时高亮正确。

## 三、阶段 1.2 交付物说明

完成 1.2 后，应满足以下内容：

### 1. 当前页高亮与设计系统一致

- **AppLayout**：侧栏 Menu 的 `selectedKeys` 根据当前路由设置（`/`、`/settings`、`/compare`、`/tasks/:id` 时 `/tasks/:id` 对应「首页」）。
- **主题**：在 `frontend/src/theme/config.ts` 中通过 `components.Menu` 显式设置 `itemSelectedColor`、`itemSelectedBg`（或等效 token），使选中项颜色与 design token 主色一致；或确认 ConfigProvider 下 Menu 选中样式已为主色且无硬编码。
- 切换路由时，侧栏对应项高亮正确，且高亮样式来自 token（与主按钮/链接主色一致）。

### 2. 品牌/标题使用 design token

- 侧栏顶部品牌区「BIM 标书生成」使用 `designTokens` 的字号、字重、颜色（如 `fontSizeHeading5`、`fontWeightStrong`、`colorText`），无硬编码色值。
- 设计系统文档（`frontend/src/theme/README.md`）中已补充「侧栏与导航（阶段 1.2）」：侧栏品牌区与 Menu 选中项 token 约定。

### 3. 「创建任务」入口（可选）

- 本阶段**不要求**在侧栏增加「创建任务」入口；保留首页内「创建任务」按钮即可。若后续需要，可在侧栏增加链接或按钮。

## 四、如何验证阶段 1.2

### 1. 验证「导航项与跳转」

1. 启动前端：`cd d:\标书工作流\frontend`，`npm run dev`。
2. 打开 http://localhost:5173/ ，侧栏应显示「首页」「设置」「对比」。
3. 点击「设置」应跳转到 `/settings`，主内容区为设置页；点击「对比」应跳转到 `/compare`；点击「首页」应回到 `/`。

### 2. 验证「当前页高亮」

1. 访问 `/`：侧栏「首页」应为选中/高亮状态。
2. 访问 `/settings`：侧栏「设置」应为选中/高亮状态。
3. 访问 `/compare`：侧栏「对比」应为选中/高亮状态。
4. 访问 `/tasks/1`（或任意任务 ID）：侧栏「首页」应为选中/高亮状态（任务详情无独立侧栏项）。
5. 高亮样式应与设计系统主色一致（如主色文字或主色浅底），无硬编码色值。

### 3. 验证「品牌/标题与 token」

- 侧栏顶部「BIM 标书生成」字号、字重、颜色与 `theme/README.md` 及 `tokens.ts` 约定一致（H5 层级、主文字色）。
- `AppLayout.tsx` 品牌区无硬编码色值，仅使用 `designTokens.xxx`。

### 4. 验收标准汇总

| 项 | 标准 |
|----|------|
| 导航项 | 侧栏内「首页」「设置」「对比」可见且点击跳转正确 |
| 当前页高亮 | `/` → 首页高亮，`/settings` → 设置高亮，`/compare` → 对比高亮，`/tasks/:id` → 首页高亮；高亮样式与设计系统一致（token 控制） |
| 品牌/标题 | 「BIM 标书生成」位于侧栏顶部，样式使用 design token |
| 创建任务 | 本阶段不要求侧栏「创建任务」入口，保留首页按钮即可 |
