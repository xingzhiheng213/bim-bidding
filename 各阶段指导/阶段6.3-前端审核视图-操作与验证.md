# 阶段 6.3 前端审核视图 — 操作与验证指南（新手向）

本文档面向新手，按「需要准备什么 → 一步步操作 → 如何验证」说明阶段 6.3 的完整流程。阶段 6.3 为**审核专家模块的前端审核视图**：在对比页 `/compare` 上扩展「从任务查看对比」（选任务、框架/章节、加载 diff）；在任务详情页的框架步骤与按章生成步骤旁增加「查看框架对比」「查看本章对比」按钮，点击后弹窗展示红绿高亮，用户无需记任务 ID 与章节号。

---

## 一、阶段 6.3 要达成什么

- **对比页扩展**：在 `/compare` 页增加「从任务查看对比」区块：选择任务（下拉）、对比类型（框架对比 / 章节对比）、章节号（选章节时），点击「加载对比」后展示 DiffView；保留「两段文本对比」区块。
- **任务详情页入口**：在框架步骤（等待审核或已确认）旁增加「查看框架对比」按钮；在按章生成完成后的每章预览旁增加「查看本章对比」按钮。点击后弹窗（Modal）内展示该任务/该章的对比（修改前/修改后原文可折叠 + 红绿 diff）。
- **可选**：对比页与任务详情弹窗中支持「编辑修改后再次对比」；已实现于对比页「从任务加载」结果下方。
- **验收标准**（来自开发计划）：选择任务与对比类型（或两段文本）能展示红绿高亮；在生成/重新生成所在界面可直接点「查看对比」无需记任务 ID 与章节号。

## 四、操作步骤（按顺序做）

### 步骤 1：在任务详情页直接查看框架对比

1. 进入一个**已完成框架**或**框架等待审核**的任务详情页（从首页任务列表点「进入」）。
2. 在「步骤 4：框架生成」区域：
   - 若框架为**等待审核**：在「接受并继续」「重新生成框架」「添加要点」同一行应看到 **「查看框架对比」** 按钮。
   - 若框架为**已确认**：在框架结果下方应看到 **「查看框架对比」** 按钮（小号）。
3. 点击 **「查看框架对比」**，应弹出 Modal，标题为「框架对比」；若该任务曾做过「重新生成框架」，会显示加载态后展示「修改前/修改后原文」折叠区与「对比结果（红=删除，绿=新增）」的 DiffView。
4. 若提示「无框架重生成前后数据」：需先对该任务点一次「重新生成框架」并等 Celery 跑完后再点「查看框架对比」。

---

### 步骤 2：在任务详情页直接查看本章对比

1. 进入一个**已完成按章生成**的任务详情页。
2. 在「步骤 5：按章生成」区域，展开「已生成 N 章，可展开预览」下的某一章（如第 1 章）。
3. 在该章的「添加要点」「重新生成本章」旁应看到 **「查看本章对比」** 按钮。
4. 点击 **「查看本章对比」**，应弹出 Modal，标题为「第 N 章对比」；若该章曾做过「重新生成本章」，会显示加载态后展示修改前/修改后原文与红绿 diff。
5. 若提示无法加载对比：需先对该章点一次「重新生成本章」并等完成后再说「查看本章对比」。

---

### 步骤 3：在对比页按任务加载对比

1. 顶部导航点击 **「对比」**，进入 http://localhost:5173/compare。
2. 页面上方为 **「从任务查看对比」**：在「任务」下拉中选择一个任务（如「任务 5 - completed」），在「对比类型」中选择「框架对比」或「章节对比」；若选「章节对比」，输入章节号（如 1）。
3. 点击 **「加载对比」**，下方应出现对比结果（含可折叠的修改前/修改后原文与红绿 DiffView）；若该任务/该章无重生成前后数据，会提示错误。
4. 页面下方为 **「两段文本对比」**：输入修改前、修改后两段文本，点击「对比」，应正常显示 diff（与 6.2 行为一致）。

## 六、常见问题

| 现象 | 可能原因 | 处理办法 |
|------|----------|----------|
| 点击「查看框架对比」后弹窗提示无法加载或 404 | 该任务从未做过「重新生成框架」 | 对该任务点一次「重新生成框架」，等完成后再说「查看框架对比」 |
| 点击「查看本章对比」后弹窗提示无法加载 | 该章从未做过「重新生成本章」 | 对该章点一次「重新生成本章」，等完成后再说「查看本章对比」 |
| 对比页「加载对比」报错 | 所选任务/章节无重生成前后数据 | 先到任务详情页对框架或该章执行一次重新生成，再回到对比页加载 |
| 任务详情页没有「查看框架对比」按钮 | 前端未更新或框架步骤未完成 | 确认 `TaskDetailPage.tsx` 已包含 6.3 修改；确认框架步骤状态为 waiting_user 或 completed |
